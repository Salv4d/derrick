# 🌍 Real World Examples (Use Cases)

Derrick was designed to solve dependency chaos in the most complex, production-grade applications on the planet. By mixing functional programming (Nix) with container networking (Docker), Derrick creates impenetrable development loops.

Below are 5 robust, production-calibre case studies (based on our internal benchmark tests) where Derrick seamlessly orchestrated intricate architectures into a localized, hermetic sandbox. Explore these topologies to master the `derrick.yaml` definitions:

1. [**Supabase**](./supabase.md) -> Managing 10+ polyglot services and aggressive `.env` validation trees.
2. [**Plausible Analytics**](./plausible.md) -> Strict boot order resolutions using Elixir OTP and ClickHouse.
3. [**Grafana**](./grafana.md) -> Isolating unified frontend/backend polyglot toolchains cleanly.
4. [**Ghost CMS**](./ghost.md) -> Pinning Node.js LTS binaries strictly without `nvm`.
5. [**Appwrite**](./appwrite.md) -> Pre-packaging horrific C++ PHP extensions (Swoole) natively in the sandbox without user compilation.
